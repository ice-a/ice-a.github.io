    SQL速查

    database SQL database
    Created At : 2023-06-06 00:00
    Count:1.3k Views 👀 :
      1. 1. 查找数据的查询
        1. WHERE: 用于过滤记录/行
        2. ORDER BY: 用于结果集的排序，升序（ASC）或者降序（DESC）
        3. SELECT TOP: 用于指定从表顶部返回的记录数
        4. LIKE: 用于搜索列中的特定模式，WHERE 子句中使用的运算符
        5. IN: 用于在 WHERE 子句中指定多个值的运算符
        6. BETWEEN: 用于过滤给定范围的值的运算符
        7. NULL: 代表一个字段没有值
        8. AS: 用于给表或者列分配别名
        9. UNION: 用于组合两个或者多个 SELECT 语句的结果集的运算符
        10. ANY|ALL: 用于检查 WHERE 或 HAVING 子句中使用的子查询条件的运算符
        11. GROUP BY: 通常与聚合函数（COUNT，MAX，MIN，SUM，AVG）一起使用，用于将结果集分组为一列或多列
        12. HAVING: HAVING 子句指定 SELECT 语句应仅返回聚合值满足指定条件的行。它被添加到 SQL 语言中，因为WHERE关键字不能与聚合函数一起使用。
    2. 2.修改数据的查询
      1. INSERT INTO: 用于在表中插入新记录/行
      2. UPDATE: 用于修改表中的现有记录/行
      3. DELETE: 用于删除表中的现有记录/行
  • 3. 聚合查询
    1. COUNT: 返回出现次数
    2. MIN() and MAX(): 返回所选列的最小/最大值
    3. AVG(): 返回数字列的平均值
    4. SUM(): 返回数值列的总和
  • 4.连接查询
    1. INNER JOIN: 内连接，返回在两张表中具有匹配值的记录
    2. LEFT (OUTER) JOIN: 左外连接，返回左表（table1）中的所有记录，以及右表中的匹配记录（table2）
    3. RIGHT (OUTER) JOIN: 右外连接，返回右表（table2）中的所有记录，以及左表（table1）中匹配的记录
    4. FULL (OUTER) JOIN: 全外连接，全连接是左右外连接的并集. 连接表包含被连接的表的所有记录, 如果缺少匹配的记录, 以 NULL 填充。
    5. Self JOIN: 自连接，表自身连接
  • 5.视图查询
    1. CREATE: 创建视图
    2. SELECT: 检索视图
    3. DROP: 删除视图
  • 6.修改表的查询
    1. ADD: 添加字段
    2. MODIFY: 修改字段数据类型
    3. DROP: 删除字段
